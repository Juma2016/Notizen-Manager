# Notes-Manager

A simple and modern notes management web application built with HTML, CSS and javascript.

## Features

- Notebook Selection: View and select from available notebooks

- Note Management: Create, view, and edit personal notes

- Quick Search: Find notes instantly by title

- Persistent Storage: Notes remain after page reload (using localStorage)
